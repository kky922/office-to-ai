# Design

Office to AI는 원본 문서를 공통 `Block` 구조로 읽은 뒤 Markdown으로 렌더링하고,
블록 경계를 유지한 채 토큰 수에 따라 청크를 나눕니다.

## 원칙

- 원본 순서와 표 구조를 가능한 한 유지한다.
- 하나의 블록은 가능하면 청크 중간에서 자르지 않는다.
- 이미지 처리는 추출, 자리표시자, 제외 중 사용자가 선택한다.
- 웹 버전은 브라우저 로컬 처리만 사용한다.
- CLI는 자동화와 대량 변환을 담당한다.

## 제한

이 도구는 문서의 의미와 텍스트 구조를 보존하는 것이 목적입니다. 픽셀 단위 레이아웃,
매크로, 애니메이션 또는 편집 가능한 원본 형식을 재현하지 않습니다.
